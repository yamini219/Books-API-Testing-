1.Description:
This repository contains API testing collections created using Postman.
It includes functional test cases for validating REST APIs such as CRUD operations, status codes, response validation, and schema checks.
The goal of this project is to ensure API reliability, correctness, and performance through structured test scenarios.

2.Tools & Technologies Used:
Postman 
JSON (Collection Format)
REST APIs

3. Please Follow the steps to Use This Project
Step 1: Clone Repository
git clone https://github.com/your-username/postman-api-testing.git
Step 2: Open Postman
Launch Postman application
Step 3: Import Collection
Click Import
Select api-collection.json
Step 4: Import Environment 
Import test-environment.json
Step 5: Run Collection
Open Collection Runner
Select environment
Click Run
